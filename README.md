Project Description
This project uses Machine Learning to predict whether a passenger would have survived the Titanic disaster based on their characteristics like class, sex, age, fare, and port of embarkation.

Features Used
1. Passenger Class (Pclass)
2. Sex
3. Age
4. Fare
5. Port of Embarkation (Embarked)

Algorithm Used
Logistic Regression

Technologies Used
- Python
- Flask
- scikit-learn
- Pandas
- NumPy
- Joblib

How to Run Locally
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open browser to `http://127.0.0.1:5000`
